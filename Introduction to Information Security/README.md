**Course:** Introduction to Information Security — Full Sail University
**Student:** Robert Potter | Office Networks and Security Wizards LLC.

---

## Course Overview

This course introduced the foundational concepts of information security, covering the threats organizations face, the controls used to defend against them, and the processes involved in building and maintaining a secure environment. Topics ranged from hands-on tool use to policy-level thinking, giving a broad view of what it means to work in a security role.

---

## What's Covered

**Threats and Vulnerabilities** — Explored common vulnerability types, malware categories, and attack vectors. Built a working understanding of how attackers operate and what defenders need to anticipate.

**Cryptography** — Covered PGP (Pretty Good Privacy) and SSL/TLS, including how public/private key encryption works, certificate trust chains, and how cryptography underpins secure communications.

**Patching and Hardening** — Hands-on work with the patch, harden, protect methodology: identifying exposure points, applying system hardening techniques, and validating that protections are in place.

**Authentication and Access Control** — Examined two-factor authentication (2FA/TFA), permission models, and the principle of least privilege as applied to user and system access.

**User Security and Awareness** — Addressed the human element of security, including social engineering, security awareness, and best practices for end-user behavior.

**Cloud Security** — Compared cloud deployment models (public, private, hybrid, community) and their respective security implications.

**Wireshark Lab** — Practical packet capture and analysis using Wireshark to observe network traffic and identify potential security concerns.

**Current Events Research** — Researched and analyzed real-world cybersecurity news, connecting classroom concepts to active threats in the field.
