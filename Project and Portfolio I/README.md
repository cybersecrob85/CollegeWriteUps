# Project and Portfolio I

**Course:** Project and Portfolio I — Full Sail University
**Student:** Robert Potter | Office Networks and Security Wizards LLC.

---

## Course Overview

Project and Portfolio I was the first capstone-style course in the Full Sail IT program. It tasked students with planning, researching, and presenting a small-scale IT solution from concept through documentation. The project centered on the Marconi Law Firm, LLC., a fictional client used as the basis for a real network and security design exercise. This course established the foundation that was built upon in Project and Portfolio II.

The course emphasized professional documentation, research-backed decision making, and the ability to translate technical plans into clear deliverables — skills that are directly applicable in a professional IT or security role.

---

## What's Covered

**IT Solution Planning** — Identified the essential functions of operating systems, application software, hardware, networking, and security practices relevant to a small business environment. Planned a solution that addressed the client's needs within realistic constraints.

**Network and Security Design** — Proposed a small-scale network infrastructure for Marconi Law Firm, including device inventory, IP schema, and initial security considerations. Documented the proposed solution in a formal project report.

**Research and Troubleshooting** — Revised the project plan based on research findings and simulated troubleshooting scenarios, demonstrating the ability to adapt designs based on new information.

**Certification and Career Awareness** — Identified continuing education and certification paths within the IT and cybersecurity industry, connecting coursework to real-world career progression.
