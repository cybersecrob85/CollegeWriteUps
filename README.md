## Capstone Projects

### Project and Portfolio I — Marconi Law Firm (IT Solution Proposal)
The first capstone course tasked me with planning and documenting a small-scale IT solution for a fictional law firm client. I produced a formal project report covering device inventory, network design, IP schema, and initial security considerations. This course established the planning and documentation foundation that carried into Project and Portfolio II.

### Project and Portfolio II — Marconi Law Firm (Full Deployment)
The second capstone was a full proof-of-concept deployment. I built a three-VM network environment from scratch — a Rocky 8 Docker host running Ghost (Node.js blog), a Rocky 8 NginX reverse proxy routing traffic to the Ghost container, and an Ubuntu LAMP stack running WordPress. I then applied a full security hardening process to WordPress covering file permissions, wp-config.php protection, and a web application firewall via the Shield plugin. The project demonstrates end-to-end skills across networking, Linux administration, containerization, web server configuration, and applied security.
