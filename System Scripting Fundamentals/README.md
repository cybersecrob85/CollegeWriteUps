**Course:** System Scripting Fundamentals — Full Sail University
**Student:** Robert Potter | Office Networks and Security Wizards LLC.

---

## Course Overview

This course introduced Python scripting with a focus on building practical, functional tools for system administration and automation tasks. The course progressed across four milestones, each adding complexity and capability to a running Python project. By the end of the course, the project evolved from basic input and output into a fully functional receipt-generating point-of-sale style application, demonstrating real scripting skills that apply directly to security automation and tooling.

---

## What's Covered

**Python Fundamentals** — Variables, data types, user input, conditionals, and loops. Built the foundation for all subsequent milestones.

**Functions and Logic** — Organized code into reusable functions, applied conditional logic to handle different user inputs and scenarios, and structured programs for readability and maintainability.

**Lists, Loops, and Data Structures** — Used lists and iteration to manage collections of items, process data dynamically, and build more complex program flows.

**File Output and Formatting** — Wrote program output to files, formatted receipt-style text output, and handled string manipulation to produce clean, readable results. The final milestone produces a formatted receipt saved to a text file.
