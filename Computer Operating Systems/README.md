## Course Overview

This course covered the fundamentals of modern operating systems with a dual focus on Linux (Ubuntu) and Windows. Labs moved from basic GUI navigation through server administration, storage management, and virtualization. It built the OS fluency that underpins the rest of the cybersecurity curriculum.

---

## What's Covered

**Linux / Ubuntu** — CLI navigation, file management, user and permission administration, and system configuration via both terminal and GUI.

**Windows** — End-user and administrative perspectives including Command Prompt, Windows Server 2019, server roles, and enterprise administration.

**Storage and Virtualization** — RAID configurations (0, 1, 5, 10), trade-offs between performance and redundancy, and an introduction to virtual machine architecture.

**Users, Groups, and Permissions** — User account management and access control across Linux and Windows, including permission models and least privilege.
