# Networking Technologies

**Course:** Networking Technologies — Full Sail University
**Student:** Robert Potter | Office Networks and Security Wizards LLC.

---

## Course Overview

This course built a foundational understanding of computer networking, covering how data moves across networks from the physical layer up through logical addressing, topology design, and real-world configuration. Labs put hands on switches, VLANs, subnetting, and network documentation — skills that translate directly into both IT operations and security roles.

---

## What's Covered

**Networking Fundamentals and the OSI Model** — Introduced networking concepts, the OSI and TCP/IP models, and how each layer contributes to data communication. Covered standards bodies and RFCs that define how protocols work.

**Physical and Logical Topology** — Designed and documented both physical (device and cable layout) and logical (IP addressing and data flow) network topologies for a small business environment. Demonstrated understanding of how physical infrastructure maps to logical network design.

**IP Addressing and Subnetting** — Hands-on subnetting lab covering classful and classless addressing, subnet masks, CIDR notation, and calculating usable host ranges. This is a core skill for any network or security role.

**ARP (Address Resolution Protocol)** — Explored how ARP maps IP addresses to MAC addresses, how ARP tables work, and the security implications of ARP in a LAN environment.

**VLANs** — Designed a VLAN segmentation scheme for a small business, demonstrating how to isolate network traffic, reduce broadcast domains, and improve security posture through logical separation.

**Network Enhancements** — Applied improvements to an existing network design, incorporating redundancy, scalability, and security considerations.
